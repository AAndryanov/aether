## Цель проекта
  
Создать бесплатную платформу для начинающих музыкантов и ди-джеев из России и СНГ, фокусирующуюся на обмене электронной музыкой, творческом сотрудничестве, наращивании аудитории и развитии сцены электронной музыки в странах постсоветского пространства.
## Оценка выполнения проекта

## Функциональные требования

#### User Stories
  
**Музыкант:**
1. *Как музыкант, я хочу загружать треки через интерфейс с минимальным количеством полей, чтобы быстро публиковать свою музыку.*  
  
Критерии приемки:
- На вкладке "Студия" при нажатии на кнопку "+", пользователь переходит на страницу загрузки, которая имеет внешний вид, схожий с заполнением формы (форма называется "Загрузить новый трек"), которая содержит поля: название, жанр (выпадающий список), поле для музыкального файла, поле для обложки. Имя артиста указывается автоматически как имя пользователя
- Пользователь может выбрать только один жанр
- При добавлении новой информации поле предпросмотра изменяется соответствующим образом
- Кнопка "Загрузить" активируется при заполнении обязательных полей (название + файл)
- Уведомление об успешной загрузке с переходом на страницу "Коллекция"
  
2. *Как музыкант, я хочу, чтобы мои треки сразу после публикации отображались в разделе TALENTPOOL, чтобы привлекать новых слушателей моего творчества*  
  
Критерий приемки:
- Трек появляется на странице "Главная" в разделе "Последние добавления" сразу после загрузки
